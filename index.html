adivinhação.py

Um jogo interativo simples onde o usuário tenta adivinhar um número aleatório. O programa fornece feedback indicando se o número procurado é maior ou menor que o palpite. É um exercício prático de controle de fluxo e lógica básica de programação.

×

adivinhaçãoevoluida.py

Versão aprimorada do jogo de adivinhação com funcionalidades extras. Inclui sistema de dicas, contagem de tentativas, níveis de dificuldade e um score baseado no desempenho. Demonstra uso avançado de estruturas condicionais, loops e variáveis para criar uma experiência mais completa e desafiadora.

×

andornot.py

Exercício prático focado nos operadores lógicos AND e NOT de Python. Explora como combinar condições usando operadores lógicos para criar expressões booleanas mais complexas. Inclui exemplos de validação de dados, verificação de múltiplas condições e negação lógica para refinar a lógica do programa.

×

arrays.py

Exercícios práticos para dominar o uso de listas (arrays) em Python. Cobre operações como adição, remoção, busca, ordenação e iteração sobre elementos. Demonstra indexação, slicing, compreensão de listas e outras técnicas essenciais para trabalhar com estruturas de dados dinâmicas.

×

elif.py

Exercícios focados em estruturas condicionais if, elif e else para criar ramificações no código. Explora como tomar decisões baseadas em múltiplas condições e direcionar o fluxo do programa. Essencial para entender como criar lógica condicional complexa e tratamento de diferentes cenários.

×

funcoes.py

Prática abrangente sobre criação, definição e utilização de funções em Python. Cobre parâmetros, argumentos nomeados, valores padrão, retorno de valores e escopo de variáveis. Demonstra como modularizar código através de funções reutilizáveis e aplicar princípios de design orientado a funções.

×

jogodavelha.py

Implementação completa do clássico jogo da velha em Python. Inclui interface com o usuário, validação de jogadas, detecção de vencedor e verificação de empate. Este projeto integra múltiplos conceitos: lógica de controle, manipulação de arrays, funções e condicionais para criar um jogo funcional e divertido.

×

mágica.py

Um fascinante truque de mágica matemática implementado em Python. O programa realiza uma série de operações matemáticas que parecem mágicas mas seguem uma lógica matemática perfeita. Demonstra como algoritmos bem construídos podem criar resultados surpreendentes e é um exercício divertido de pensamento lógico.

×

notas.py

Sistema prático para gerenciar e calcular notas de alunos. Implementa funcionalidades como armazenar múltiplas notas, calcular médias, determinar aprovação/reprovação, e gerar relatórios. Aplicação real que integra listas, loops, funções e cálculos matemáticos para resolver um problema do mundo real.

×

somadenumeros.py

Exercício fundamentado em operações aritméticas com números em Python. Explora soma, subtração, multiplicação e divisão, bem como operações mais complexas com múltiplos números. Inclui validação de entrada e cálculos de estatísticas básicas, formando uma base sólida para compreender manipulação de dados numéricos.

×

while.py

Exercícios práticos com loops while e iterações em Python. Cobre como criar repetições baseadas em condições, evitar loops infinitos com condições de parada apropriadas e usar variáveis contadoras. Essencial para dominar a repetição controlada de código e entender quando usar while versus outras estruturas de iteração.

×

simuladordecompras.py

Exercício simples para interpretar e entender melhor erros de sintaxe, sintáticos e léxicos de forma prática, não sendo algo extremamente complexo, entretando o suficiente para aplicar o conhecimento de interpretação de erros.
